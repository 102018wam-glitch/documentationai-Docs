## Prerequisites

Before you begin, ensure you have:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An active email address for account verification
- Optional: A GitHub account for version control integration

</Callout>

No software installation required—everything runs in your browser.

## Create Your First Documentation Project

Organize your project docs immediately after login.

<Steps>
  <Step title="New Project" icon="plus">
    Click **New Project** in the top-right corner. Enter a name like `My First Docs` and select a template (e.g., "Default").
  </Step>
  <Step title="Add Your First Page" icon="file-text">
    In the project sidebar, select **Pages** > **New Page**. Title it `Introduction` and add content using the MDX editor.
  </Step>
  <Step title="Publish" icon="globe">
    Hit **Publish** to make your docs live. Share the generated URL with your team.
  </Step>
</Steps>

<CodeGroup tabs="MDX,Markdown">
````mdx
## Welcome to My Docs

This is your first page powered by Валерий Medvedev.

- **Feature 1**: Real-time collaboration
- **Feature 2**: MDX support with components
````
