## Overview

Валерий Медведев provides a centralized space to create, organize, and manage your project documentation. You can build structured docs sites with MDX support, version history, and search capabilities. Use it to streamline collaboration across your team and deliver polished documentation to users.

<Callout kind="info">
  Валерий Медведев uses brand color `<#3B82F6>` for primary elements like buttons and links.
</Callout>

## Quick Start

Get up and running with Валерий Медведев in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com` using your email.
  </Step>
  <Step title="New Space" icon="plus">
    Click "New Documentation Space" and name it "Валерий Медведев".
  </Step>
  <Step title="Add Pages" icon="file-plus">
    Create your first page with the MDX editor.
  </Step>
  <Step title="Publish" icon="globe">
    Deploy your site to a custom domain.
  </Step>
</Steps>

<CodeGroup tabs="CLI,API">
  ```bash
  # Install CLI
  npm install -g valery-medvedev-cli

  # Initialize space
  valery init my-docs --template docs
  ```
  ```javascript
  // API example
  const response = await fetch('https://api.example.com/spaces', {
    method: 'POST',
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` },
    body: JSON.stringify({ name: 'Валерий Медведев' })
  });
  ```
</CodeGroup>
