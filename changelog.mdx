## Upgrading to the Latest Version

Follow these steps to update your Валерий Медведев instance.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export all documents via the dashboard.
  </Step>
  <Step title="Update Package" icon="download">
    
````bash
npm install valery-medvedev@latest
````
    
  </Step>
  <Step title="Run Migrations" icon="settings">
    
````bash
npm run migrate
````
    
  </Step>
  <Step title="Restart Server" icon="refresh-cw">
    Restart your application and verify changes.
  </Step>
</Steps>

## Example: New Real-Time API Usage

Demonstrates the new collaboration endpoint in v1.3.0.

<Tabs>
  <Tab title="JavaScript" icon="code">
    
````javascript
const response = await fetch('https://api.example.com/v2/collaborate', {
  method: 'POST',
  headers: { 'Authorization': `Bearer ${YOUR_TOKEN}` },
  body: JSON.stringify({ docId: 'doc-123', userId: 'user-456' })
});
const session = await response.json();
console.log(session.url);
````
  </Tab>
  <Tab title="Python" icon="python">
    
````python
import requests

response = requests.post(
    'https://api.example.com/v2/collaborate',
    headers={'Authorization': f'Bearer {YOUR_TOKEN}'},
    json={'docId': 'doc-123', 'userId': 'user-456'}
)
session = response.json()
print(session['url'])
````
  </Tab>
</Tabs>
